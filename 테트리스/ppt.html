Canvas 이용한 도형 그리기

1. 그리드와 블럭
캔버스 요소 위에 30px 간격의 그리드를 그려 테트리스 보드를 구성했습니다.
각 블럭은 2차원 배열로 구성되어 있고, drawTetrisBlock() 함수를 통해 실제 캔버스에 색상 또는 이미지로 렌더링됩니다.

블럭 : fillRect(x, y, width, height)
context.fillStyle=color 색칠된 직사각형을 그리기

2. 블럭 랜덤 생성 및 조작
화살표 키로 블럭을 좌우 이동, 아래로 내리기, 위로 회전이 가능합니다.
새 블럭은 move() 함수에서 바닥 또는 다른 블럭과 충돌할 때마다 새로 생성됩니다.
move(): 충돌 검사 → 고정 또는 제거 → 새 블럭 생성
rotate(): 배열 전치 → 회전

3. 폭탄
폭탄 블럭은 이미지로 표현되고, 블럭에 닿으면 3x3 범위의 블럭을 모두 제거합니다.
기존 블럭과의 차별화를 위해 BOMB_INDEX를 따로 설정하고, move() 함수에서 특수 동작을 처리합니다.
